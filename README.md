# Facial Expression Recognition using CNN

## Project Description

This project develops a CNN-based Facial Expression Recognition system using the FER2013 dataset.

The model classifies facial expressions into:

* Angry
* Disgust
* Fear
* Happy
* Neutral
* Sad
* Surprise

## Dataset

FER2013 Dataset

https://www.kaggle.com/datasets/msambare/fer2013

* Matplotlib

## How to Run

1. Download FER2013 dataset.
2. Open Kaggle Notebook.
3. Upload dataset.
4. Run all cells.
5. Generated figures will be saved automatically.
